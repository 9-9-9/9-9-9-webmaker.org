Explore creative ways to teach digital superpowers

Get involved with Webmaker through fun making and sharing, backed by the global Mozilla community’s Web Literacy Map. It’s all part of our non-profit mission to create a more open, web literate world.

A cookbook for teaching the web

Each skill includes free recipes, resources and teaching kits anyone can use to teach others to understand how the web works. Discover new powers like coding, storytelling, protecting your online privacy and more.

Free tools that make it easy to create and learn

Webmaker tools make it easy to create things on the web. Make your own web pages, interactive videos, remixes, mobile apps and more—learning web mechanics, code and other valuable skills as you go.

Make. Learn. Get together.

The Webmaker community gets together at events around the world—at libraries, community centers, internet cafes, or in your own living room. Find an event near you, learn how to host your own, or get involved with our global Maker Party (July to Sep 2014).
